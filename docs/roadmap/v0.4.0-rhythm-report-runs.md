## Reading

- Global summaries mix multiple groups, so beat stability is expected to drop
  when metric, entrained, and flow layers are present.
- Group-level summaries are better for checking an isolated pulse surface; group
  2 in both metric and flagship candidates stays near the expected stable-beat
  range.
- Population synchronization is now visible in the global Kuramoto summaries.
  The order is measured after removing each voice's phase offset, so it reflects
  relative entrainment rather than absolute randomized oscillator phase.
- Flow candidates show positive burstiness and low beat-stability, which is the
  main report guardrail for a non-metric texture. Slope fields remain diagnostic
  only; the current flow implementation no longer tries to force direct
  long-memory 1/f IOI drift.
- After audition feedback, flow timing was changed from slow rate warping and
  direct pink IOI drift into a stationary renewal process with local jitter and
  short rain-like clusters. Dense non-binary clock points now only provide
  onset resolution.
