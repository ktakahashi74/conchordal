# ドキュメント

## Conchordalでの作曲

Conchordalにおける作曲とは**シナリオスクリプティング**である。それは、自律的な音のエージェントが認知ランドスケープを探索し、自ら調和を発見していく時間的構造を設計する行為である。従来の作曲が五線譜やピアノロールといった記号的グリッドへ音を配置するのに対し、Conchordalのスクリプティングは、エージェントが*いつ*、*どのように*ランドスケープへ生み出されるかを定義し、音符を一つ一つ指定するのではなく、相互作用と創発を通じて協和を見出すことを可能にする。

シナリオは**Rhai**という軽量スクリプト言語で記述される。Rhaiを通じて、音の時間的展開を振り付ける。エージェントの種を定義し、特定の時刻にグループを生成し、初期パラメータを設定し、ランドスケープの物理法則を変調する。スクリプトは固定された音符の楽譜ではなく、生成的プロセスの楽譜となる—音楽的創発のための設計図である。
