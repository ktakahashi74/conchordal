# ドキュメント

## Conchordalでの作曲

Conchordalの作曲は**シナリオスクリプティング**。**Rhai**スクリプトで、音のエージェントをいつ、どのようにランドスケープへ生成するかを記述する。音符を直接配置するのではなく、エージェントのパラメータを設定し、ランドスケープ上での創発的な相互作用を通じて協和を見出させる。

- [シナリオスクリプティングリファレンス](/docs/rhai/)
